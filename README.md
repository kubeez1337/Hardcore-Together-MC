
# Hardcore Together


## Key Features

* **Synchronized Health & Hunger Pools**: Perfect real-time synchronization of health, hunger levels, and saturation across all online players. 
* **Cooperative Permadeath**: When any player dies, the run ends. The plugin automatically locks down the world, kills remaining players, shows a game-over screen, and wipes world directories to generate a fresh seed on restart.
* **Persistent Speedrun Stopwatch**: A lightweight, in-memory stopwatch displayed on player Action Bars. It automatically saves to disk to survive server restarts, pausing when the server is empty or the game is over, and resetting only when a new run begins.
* **All-Time Black Points Leaderboard**: A sidebar display ranking players by how many failures they've caused over the lifetime of the server. Includes OP commands to adjust points.
* **Lifetime Failure Statistics**: Persistent tracking of total failed runs and individual failure counts.
* **Message of the Day (MOTD)**: A welcoming info screen shown to players when they join, detailing command guides and stopwatch status.
* **Operator Force Reset**: A command to let server operators forcefully reset the world and restart the game at any point.

* **Real-time Event Logging**: Formatted alerts detailing final hits (e.g., Creeper explosions, fall damage).
* **Speedrun Stopwatch**: Displays the exact duration of the run.
* **Detailed Run Statistics**: Broad-scope metrics (damage dealt, damage taken in hearts, and hunger restored) per player for the run.
* **Seamless World Reset**: Clickable `[ ▶ PLAY AGAIN ]` button for players to instantly wipe worlds, generate a new seed, and restart the server.
* **Black Points Sidebar**: Live rankings tracking who is responsible for run failures.
